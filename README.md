# mechmatch

A minimalist browser-based quiz game where you identify which branch of classical or quantum mechanics each equation belongs to.

## how to play

Open `physics-guessing-game.html` in any modern browser. No build step or server required.

Each round displays a rendered equation. Select the framework you think it belongs to. You have 5 lives, a wrong answer costs one. The game ends when all lives are lost.

## features

- 27 equations across four mechanics frameworks
- Live score tracking and heart-based life system
- LaTeX equation rendering via MathJax
- Notion-inspired minimal UI

## stack

Plain HTML, CSS, and JavaScript. MathJax CDN for equation rendering. No dependencies to install.
